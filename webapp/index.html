BioIT Genome Intelligence

AWS data lake pipeline for human genome analysis.

An end-to-end data pipeline that pulls chromosome sequences from the NCBI API, processes them through a high-performance C++ FASTA parser running on Lambda and AWS Batch, and lands Parquet datasets in an S3 data lake — queryable via Athena and visualized here.
