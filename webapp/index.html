Glossary

Bioinformatics terms used in the dashboard

Short definitions for the biology and data terms used throughout the dashboard.

Chromosome

A large DNA molecule that carries part of the genome. Humans typically have chromosomes 1-22 plus X and Y.

Gene

A DNA region that can be transcribed into RNA and often translated into a protein or other functional product.

ORF

Open reading frame: a stretch of sequence that starts like a protein-coding region and continues without an in-frame stop codon.

Candidate coding region

A genomic window flagged as potentially protein-coding because it contains one or more ORFs and supportive sequence signals.

CpG

A cytosine followed by guanine on the same DNA strand. CpG-rich areas are often biologically important for gene regulation.

GC content

The percentage of bases in a region that are G or C. GC-rich regions can have different structural and functional properties than AT-rich regions.

Motif

A short recurring DNA pattern that may indicate a biological function, such as regulation, transcription, or sequence processing.

Repeat

A sequence pattern that occurs multiple times. Repeats can be short, long, or low-complexity and may affect genome structure and analysis difficulty.

Homopolymer

A run of the same base repeated many times, such as `AAAAAA` or `TTTTTT`.

Start codon

A sequence, usually `ATG`, that commonly marks the start of a protein-coding reading frame.

Polyadenylation signal

A short motif, commonly `AATAAA`, associated with RNA processing and transcript termination.

Genomic window

A fixed-size slice of the chromosome used for regional summaries like GC%, motif density, and ORF counts.

Gene annotation

Reference metadata describing known genes, transcripts, exons, and related features from sources like NCBI or Ensembl.

Ideogram

A simplified visual representation of a chromosome used here to show chromosome-scale context and analysis hotspots.

Parquet

A columnar data file format optimized for analytics. The pipeline stores outputs in Parquet so they can be queried efficiently.

Athena

A serverless AWS SQL query engine used to read the Parquet datasets directly from S3.
